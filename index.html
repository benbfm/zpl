🚀 First-Time Setup (do once)初次配置（仅需一次）

      
1

        Design your label — Go to ✏️ Designer, set the label size, add text, barcodes, and boxes to build the layout.
        设计标签 — 前往✏️ 设计器，设置标签尺寸，添加文本、条码和方框，构建标签布局。
        
Tip: mark fields that change each print (like order number or product name) as Variables so they can be filled in at print time.提示：将每次打印都会改变的字段（如订单号或产品名称）设为变量，以便打印时逐次填写。

      

      
2

        Save the template — Enter a name and click 💾 Save Template. Your design is stored in the browser's local storage and available on the 📂 Templates screen.
        保存模板 — 输入名称后点击💾 保存模板。设计将存储在浏览器本地，并可在📂 模板库界面中查看。
      

      
3

        Set your printer URL — Go to 🖨️ Print and enter the Zebra printer's network address in the Printer URL field. Example: http://192.168.1.100:9100
        配置打印机地址 — 前往🖨️ 打印，在打印机地址字段中输入斑马打印机的网络地址，例如：http://192.168.1.100:9100
        
The URL is saved automatically — you only need to enter it once. Ask your IT team for the correct IP and port.地址会自动保存，只需输入一次。请向 IT 团队确认正确的 IP 和端口号。

      


      
📅 Daily Workflow — Printing Labels日常工作流程 — 打印标签

      
A

        Quick print (known values): Go to 🖨️ Print → select template → fill in variable values → click ⚡ Generate ZPL → click 🖨 Send to Printer.
        快速打印（已知内容）： 前往🖨️ 打印 → 选择模板 → 填写变量值 → 点击⚡ 生成 ZPL → 点击🖨 发送到打印机。
      

      
B

        Scan-to-print (order fulfilment): Go to 📷 Scan & Print → scan the order barcode with a USB gun or camera → the app looks up the order and fills the template automatically → click Proceed to Print →.
        扫描打印（订单发货）： 前往📷 扫描打印 → 使用 USB 扫描枪或摄像头扫描订单条码 → 应用自动查询订单并填写模板 → 点击前往打印 →。

📐 Step 1 — Set the Label Size第 1 步 — 设置标签尺寸

      
1

        In the left sidebar under Label Size, choose a Preset from the dropdown (common Zebra sizes) or enter Width and Height manually in dots.
        在左侧边栏标签尺寸区域，从下拉列表选择预设尺寸（常用斑马尺寸），或手动输入宽度和高度（单位：点）。
        
At 203 DPI: 8 dots ≈ 1 mm. Example — 4×2 inch label = 812×406 dots.203 DPI 下：8 点 ≈ 1 毫米。示例：4×2 英寸标签 = 812×406 点。

      


      
➕ Step 2 — Add & Position Elements第 2 步 — 添加并放置元素

      
1

        Click an Add Element button in the left sidebar. Choose the type that fits your need:
        点击左侧边栏中的添加元素按钮，选择适合的类型：
        

          🅃 Text — static or variable text line  |  ▐▌ Code 128 — general-purpose 1D barcode  |  ⊞ QR Code — 2D code (URLs, long text)  |  ||| EAN-13 — retail barcode (13 digits)  |  ||| UPC-A — retail barcode (12 digits)  |  □ Box — border / divider line
          🅃 文本 — 静态或变量文字  |  ▐▌ Code 128 — 通用一维条码  |  ⊞ 二维码 — 二维码（网址、长文本） |  ||| EAN-13 — 零售条码（13位） |  ||| UPC-A — 零售条码（12位） |  □ 方框 — 边框 / 分隔线
        

      

      
2

        Drag the element on the white label canvas to position it. You can also type exact X and Y coordinates (in dots) in the right-side properties panel.
        在白色标签画布上拖动元素调整位置，也可在右侧属性面板直接输入精确的 X、Y 坐标（单位：点）。
      

      
3

        Resize by dragging the small handle at the bottom-right corner of a selected element, or edit W / H in the properties panel.
        拖动选中元素右下角的小控制柄可以调整大小，也可直接在属性面板中修改宽 / 高值。

🖨️ How to Print a Label (Step-by-Step)如何打印标签（分步说明）

      
1

        Click 🖨️ Print in the navigation bar to open the Print screen.
        点击导航栏中的🖨️ 打印，打开打印界面。
      

      
2

        Use the Select Template dropdown to choose the label design you want to print.
        使用选择模板下拉菜单，选择要打印的标签设计。
        
If the list is empty, go to ✏️ Designer first to build and save a template.如果列表为空，请先前往✏️ 设计器创建并保存模板。

      

      
3

        If the template has variable fields, an input box appears for each one. Type the value for this print job — for example the order number, batch code, or product name.
        如果模板包含变量字段，每个变量都会显示一个输入框，填写本次打印的内容，例如订单号、批次码或产品名称。
        
Variable fields were created in the Designer by enabling Mark as Variable on an element. They appear as {VariableName} in the label preview.变量字段是在设计器中为元素启用标记为变量后创建的，在标签预览中显示为 {变量名称}。

      

      
4

        Click ⚡ Generate ZPL. The ZPL code is built and shown in the output area. The label preview updates to show the filled-in values.
        点击⚡ 生成 ZPL，ZPL 代码生成并显示在输出区域，标签预览同时更新为填写后的内容。
      

      
5

        Check the Printer URL field. It should show the Zebra printer's network address. If it is empty, type it in — e.g. http://192.168.1.100:9100
        检查打印机地址字段，应显示斑马打印机的网络地址。若为空，请填入地址，例如：http://192.168.1.100:9100
        
The address is saved automatically in your browser after the first entry — it will be remembered for future sessions.首次填写后，地址将自动保存在浏览器中，后续会话无需重复输入。

      

      
6

        Click 🖨 Send Raw ZPL to Printer. The app transmits the ZPL directly to the printer. A green success message confirms the label was printed.
        点击🖨 发送原始 ZPL 到打印机，应用将 ZPL 直接发送至打印机，绿色成功提示确认标签已打印。
      


      
📥 Saving / Copying the ZPL File保存 / 复制 ZPL 文件

      
📋

        Click 📋 Copy ZPL to copy the ZPL code to your clipboard — paste it into any ZPL tool or terminal.
        点击📋 复制 ZPL 将代码复制到剪贴板，可粘贴到任意 ZPL 工具或终端。
      

      
💾

        Click 💾 Download ZPL to save the code as a .zpl text file — useful for batch printing via USB or network share.
        点击💾 下载 ZPL 将代码保存为 .zpl 文本文件，适合通过 USB 或网络共享进行批量打印。
      


      
🔢 Quantity Printing数量打印

      
Q

        Use the Quantity field to print multiple copies of the same label in a single send. The ZPL ^PQ command is added automatically.
        使用数量字段，单次发送可打印同一标签的多个副本，系统会自动添加 ZPL 的 ^PQ 命令。
      


      
🌐 Network requirement:网络要求：
         The Zebra printer must be connected to the same network as your computer. The standard raw ZPL port is 9100. Contact your IT team if you are unsure of the printer's IP address.
         斑马打印机须与您的电脑处于同一网络。原始 ZPL 标准端口为 9100。如不确定打印机 IP 地址，请联系 IT 团队。
      

      
⚠️ If the printer does not respond:如果打印机无响应：
         Check that (1) the printer is powered on, (2) the network cable or Wi-Fi is connected, (3) the IP address in Printer URL is correct, and (4) port 9100 is not blocked by a firewall.
         请检查：(1) 打印机是否开机；(2) 网线或 Wi-Fi 是否连接；(3) 打印机地址中的 IP 是否正确；(4) 防火墙是否拦截了 9100 端口。
      

    


    
    

      
📷 How to Scan & Print (Step-by-Step)如何扫描并打印（分步说明）

      
1

        Click 📷 Scan & Print in the navigation bar.
        点击导航栏中的📷 扫描打印。
      

      
2

        Scan the order barcode using one of three methods:
        扫描订单条码，可选以下三种方式之一：
        

          
            📡 USB Barcode Scanner (recommended): Click the Order # / Barcode input box to give it focus, then pull the trigger on your USB scanner gun. The barcode value is typed in automatically and the lookup runs immediately.


            📷 Camera: Click ▶ Start Camera and point the camera at the barcode — detection happens automatically when a valid barcode is in frame. Works on Chrome, Edge, and Android Chrome.


            ⌨️ Manual: Click the input box, type the order number, and press Enter.
          
          
            📡 USB 扫描枪（推荐）： 点击订单号 / 条码输入框使其获得焦点，然后扣下扫描枪扳机，条码值将自动填入并立即查询。


            📷 摄像头： 点击▶ 开始摄像，将摄像头对准条码，检测到有效条码后自动识别。支持 Chrome、Edge 和 Android Chrome。


            ⌨️ 手动输入： 点击输入框，输入订单号，按 Enter。
          
        

      

      
3

        The app looks up the order and displays the order details in the Order Info panel — SKU, product name, quantity, and customer.
        应用查询订单，并在订单信息面板中显示详细内容，包括 SKU、产品名称、数量和客户信息。
      

      
4

        The Label Template is selected automatically if the SKU matches a linked template. Otherwise, choose the correct template from the dropdown.
        如果 SKU 与已关联的模板匹配，标签模板将自动选中；否则请从下拉菜单手动选择正确的模板。
        
To link a template to a SKU: open the template in the Designer, type the SKU in the Linked SKU field, and save.关联模板与 SKU 的方法：在设计器中打开该模板，在关联 SKU 字段输入 SKU，保存即可。

      

      
5

        Check the Variable Mapping panel — the app has auto-filled each variable field from the order data. Adjust any value if needed.
        检查变量映射面板，应用已从订单数据中自动填写各变量字段，如需调整可手动修改。
      

      
6

        Click Proceed to Print →. The app opens the 🖨️ Print screen with all variable values pre-loaded.
        点击前往打印 →，应用跳转至🖨️ 打印界面，所有变量值已预先填写完毕。
      

      
7

        Click ⚡ Generate ZPL then 🖨 Send Raw ZPL to Printer. The label prints immediately.
        点击⚡ 生成 ZPL，再点击🖨 发送原始 ZPL 到打印机，标签立即打印。
